Seja bem-vindo ao curso de Projetos de JavaScript!


No curso de JavaScript Aplicado a Projetos, você praticará a essência da web, explorando HTML, CSS e JavaScript através da execução prática de projetos.


Os projetos abrangem um leque de complexidade variada, permitindo-lhe desenvolver desde aplicações básicas até as mais elaboradas, tanto em termos de design quanto em lógica de programação, onde os desafios algorítmicos serão altos, importante para o desenvolvimento de programadores.


Você aprenderá seguindo padrões de qualidade profissionais, adquirindo experiência real sobre como os desenvolvedores experientes estruturam seus códigos e, ao mesmo tempo, construirá um portfólio robusto com mais de 20 projetos em JavaScript.


Com a crescente demanda por JavaScript no universo do desenvolvimento de software, dominar essa linguagem se torna decisivo para quem aspira a posições de destaque em empresas nacionais e internacionais, tanto em desenvolvimento front-end quanto back-end.


Este curso simula a rotina profissional de um Desenvolvedor, sendo uma oportunidade imperdível para quem busca entrar no mercado de trabalho.


Além disso, o JavaScript é fundamental para quem deseja se aprofundar em bibliotecas e frameworks modernos e em demanda, como React, Vue, Angular, Svelte, entre outros. Se você tem dificuldades com essas tecnologias ou quer começar a aprendê-las, este curso oferece toda a base necessária para avançar no ecossistema de frameworks JavaScript.


Os pré-requisitos são conhecimentos básicos de HTML, CSS e JavaScript. Ter realizado algum projeto nessas linguagens já é um bom começo (se você deseja aprofundar seus conhecimentos nessas áreas, nossos outros cursos na plataforma podem ajudar).


Um entendimento básico do DOM é essencial, pois ele é a chave para manipular elementos HTML e estilos CSS em websites.

O que é HTML e CSS?


HTML (Linguagem de Marcação de Hipertexto) é a fundação da web, uma linguagem de marcação responsável por estruturar o conteúdo e definir a base de qualquer página na internet. Utilizando HTML, criamos a estrutura de um site por meio da definição de elementos como títulos, parágrafos, imagens, links e muitos outros, organizando assim o conteúdo que será exibido aos usuários. CSS (Folhas de Estilo em Cascata), complementa o HTML ao proporcionar estilização visual às páginas web.


Com CSS, é possível customizar o design, ajustando cores, fontes, espaçamentos, responsividade e layout, entre outros aspectos visuais, para criar experiências de navegação atrativas e coerentes com a identidade visual desejada. No contexto do desenvolvimento web, o domínio de HTML e CSS é essencial, pois juntos, eles formam a base para a construção de websites funcionalmente ricos e visualmente impressionantes. Este curso oferece uma imersão profunda nessas tecnologias, capacitando você a criar desde sites simples e elegantes até complexas interfaces de usuário web.


O que é JavaScript?


JavaScript é uma linguagem de programação interpretada essencial para o desenvolvimento web moderno. Conhecida por sua capacidade de adicionar interatividade e dinamismo às páginas da web, JavaScript permite a criação de conteúdo atualizável, animações, jogos, aplicativos web complexos, e muito mais. É a linguagem por trás da lógica de programação do cliente, interagindo com HTML e CSS para criar experiências de usuário ricas e envolventes.


Dominar JavaScript é importante para qualquer desenvolvedor web que deseja criar aplicações web avançadas e responsivas, e nosso curso está projetado para equipá-lo com as habilidades necessárias para utilizar JavaScript efetivamente em seus projetos.


O que é DOM?


O DOM (Document Object Model) é uma interface de programação para documentos web. Ele representa a página de modo que programas podem alterá-la e manipulá-la. Em essência, o DOM transforma os elementos de um documento HTML em objetos que podem ser manipulados com JavaScript. Isso permite que os desenvolvedores interajam e modifiquem dinamicamente o conteúdo, a estrutura e o estilo das páginas da web, respondendo a interações do usuário, atualizações de conteúdo e outros eventos em tempo real.


Saber como manipular o DOM é fundamental para criar aplicações web interativas e dinâmicas, e este curso cobre extensivamente como usar o DOM juntamente com JavaScript para enriquecer suas aplicações web.
